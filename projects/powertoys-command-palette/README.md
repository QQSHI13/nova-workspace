# Quick Notes - PowerToys Command Palette Extension

A simple PowerToys Command Palette extension for Quick Notes.

## Features

- **Open Notepad**: Creates a new timestamped note in `~/notes/` and opens it in Notepad
- **Settings**: Coming soon!
- **Help**: Shows quick help information

## Installation

1. Build the project:
   ```bash
   dotnet build -c Release
   ```

2. Copy the output DLL to your PowerToys plugins directory

3. Restart PowerToys

## Usage

Open PowerToys Command Palette (default: `Alt+Space`), then type:
- "Open Notepad" to create a new note
- "Settings" for settings (coming soon)
- "Help" for help
