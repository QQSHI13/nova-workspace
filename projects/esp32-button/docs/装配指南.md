## 焊接步骤

### 第一步: SMD 小元件 (电阻电容)

**技巧**: 先在一个焊盘上锡，然后用镊子固定元件，再焊另一端。

1. **电阻** (R1, R2 1kΩ; R3 1.2kΩ; R4 100Ω)
   - 0603 封装很小，用镊子辅助
   - 注意方向: 电阻无极性

2. **电容** (C1, C2 10µF)
   - 0603 陶瓷电容无极性
   - 如果是有极性的钽电容，注意正负极

### 第二步: LED

**极性很重要!**
- 红色 LED: CHRG (充电中)
- 绿色 LED: STDBY (充满)
- 0603 LED 有标记的一端是负极 (通常有绿点或切口)

### 第三步: IC 芯片

**TP4056 (SOP-8)**
```
引脚顺序 (顶视图，缺口在左):
     ┌──────┐
  1  │      │  8
  2  │  TP  │  7
  3  │ 4056 │  6
  4  │      │  5
     └──────┘

1: TEMP  (可不接)
2: PROG  → 1.2kΩ → GND
3: GND   → GND
4: VIN   → USB 5V
5: BAT   → 电池 (经过保护电路)
6: STDBY → 绿色 LED (1kΩ) → VCC
7: CHRG  → 红色 LED (1kΩ) → VCC
8: CE    → VCC (使能)
```

**XC6206 (SOT-23)**
```
引脚 (顶视图):
   ┌───┐
 1 │   │ 3
    │   │
     └─2─┘

1: VIN  → 电池
2: VOUT → 3.3V 输出
3: GND  → GND
```

**DW01A + FS8205A (如果使用)**
- 参考 schematic.md 的连接方式
- 这两个是保护电路，建议添加

### 第四步: 连接器

1. **USB-C 母座**
   - 贴板焊接，注意方向
   - VBUS → TP4056 VIN
   - GND → GND

2. **JST-PH 2P (电池接口)**
   - 直插焊接
   - 注意正负极标记

3. **HY2.0-4P (GROVE 连接器)**
   - 方向: 缺口朝向板内
   - 引脚对应关系:
     - Pin 1 (黑): GND
     - Pin 2 (红): 3.3V (LDO 输出)
     - Pin 3 (黄): G26 → 蜂鸣器
     - Pin 4 (白): G32 (备用)

4. **滑动开关**
   - 3 个引脚: 中间是公共端
   - 连接: 电池正极 → 开关 → TP4056 BAT

### 第五步: 蜂鸣器

- 有源蜂鸣器 (内置振荡电路)
- 正极 → G26 (经过 100Ω 电阻)
- 负极 → GND
- 如果声音太小，可以去掉电阻

## 测试 (焊接完成后，连接 Atom Lite 前)

### 万用表检查

1. **短路检查**
   - 测量 3.3V 和 GND 之间: 应无短路 (无穷大)
   - 测量 5V 和 GND 之间: 应无短路

2. **LDO 输出测试**
   - 用电池或 USB 供电
   - 测量 LDO VOUT: 应为 3.3V

3. **充电测试**
   - 连接 USB
   - 红色 LED 应亮起 (充电中)

### Atom Lite 连接

1. 确保开关处于 OFF 位置
2. 对准 GROVE 连接器，插入 Atom Lite
3. 打开开关
4. Atom Lite 的 LED 应亮起 (蓝色或白色)

## 固件烧录 (首次)

1. **连接 USB** 到电脑
2. **安装驱动** (如果需要)
   - Windows 可能需要 FTDI VCP 驱动
3. **打开 PlatformIO**
4. **编译上传**
   ```bash
   pio run --target upload
   ```
5. **打开串口监视器**
   ```bash
   pio device monitor
   ```
6. 应该看到启动信息:
   ```
   === Pomodoro Button v1.0 ===
   Settings: work=25min, break=5min...
   ```

## 外壳组装 (Unified Case 设计)

### 装入电池

1. 将 503040 电池放入下盖电池槽
2. 连接 JST 插头到扩展板
3. 确保电池线缆有足够空间

### 安装扩展板

1. 将焊接好的扩展板放入中层支架
   - **注意**: 元件面朝下 (朝向电池)
   - USB-C 对准侧面开口
   - 开关对准侧面开口

### 连接 Atom Lite

1. 将 GROVE 4P 排线一端插入扩展板
2. 另一端插入 Atom Lite 的 GROVE 接口
3. 将 Atom Lite 放入上层位置
   - 按钮朝上
   - USB-C 对准顶部开口

### 固定螺丝

1. 将 M2x25mm 螺丝从顶部中心孔插入
2. 穿过 Atom Lite、扩展板的 M2 孔
3. 在底部用 M2 螺母拧紧
   - 或用自锁螺母防止松动

### 盖上顶盖

1. 将顶盖对准卡扣位置
2. 按压直到听到 "咔" 声
3. 确保顶盖牢固

## 故障排除

### Atom Lite 不亮
- 检查电池是否有电
- 检查开关是否打开
- 测量 GROVE Pin 2 电压 (应为 3.3V)
- 检查 LDO 是否正常工作

### 无法充电
- 检查 USB-C 焊接
- 检查 TP4056 连接
- 测量 TP4056 BAT 脚电压

### 蜂鸣器不响
- 检查蜂鸣器极性
- 检查 G26 连接
- 在固件中开启声音

### 同步失败
- 确保长按进入同步模式 (蓝色旋转)
- 检查 USB 连接
- 使用 Chrome/Edge 浏览器

## 同步设置

### 方法 1: 使用串口监视器 (推荐)

1. 长按按钮 3 秒进入同步模式 (LED 蓝色旋转)
2. 连接 USB 到电脑
3. 打开 PlatformIO 串口监视器:
   ```bash
   pio device monitor
   ```
4. 输入设置命令:
   ```
   SET:work=25,break=5,longBreak=15,sessions=4,sound=1
   ```
5. 设备回复 "OK" 表示设置成功

### 方法 2: 使用网页界面

1. 长按按钮进入同步模式 (蓝色旋转)
2. 用 Chrome/Edge 打开 `web-sync/index.html`
3. 点击 "Connect Button"
4. 调整设置并点击 "Sync"

### 设置命令格式

```
SET:work=25,break=5,longBreak=15,sessions=4,sound=1
```

- `work=25` - 工作时长 (分钟, 1-60)
- `break=5` - 短休息时长 (分钟, 1-30)
- `longBreak=15` - 长休息时长 (分钟, 5-60)
- `sessions=4` - 长休息前的工作会话数 (2-10)
- `sound=1` - 声音开关 (0=关, 1=开)

## 安全注意事项

⚠️ **锂电池安全**
- 不要刺穿或挤压电池
- 不要短路电池正负极
- 充电时使用合格充电器
- 如果发现电池鼓包，立即停止使用

⚠️ **焊接安全**
- 烙铁温度不要过高 (建议 320°C)
- 焊接时间不要过长 (每个点 < 3 秒)
- 在通风良好的环境工作
