☄️ Button Sync

Sync settings with your Pomodoro Button

Not connected. Long-press button to enter sync mode, then connect USB.
Requires Chrome/Edge with Web Serial API support.
LED Colors:
Red = Work
Green = Short Break
Blue = Long Break
How to sync:
  1. Long-press (3s) the button until LED turns blue
  2. Connect USB cable to your PC
  3. Click "Connect Button" above (requires Chrome/Edge)
  4. Adjust settings and click "Sync"

Alternative: Use Arduino IDE or PlatformIO serial monitor at 115200 baud.
