# 🔋 Battery Panic

> A pixel art game about surviving low battery emergencies while learning real battery-saving tips!

## 🎮 Game Concept

You're in a race against time with only **15% battery** and **20 minutes** to complete critical tasks before your laptop dies. Every decision matters - close Chrome tabs, dim the screen, or risk losing everything!

## ✨ Features

### 📊 Core Mechanics
- **Resource Management**: Balance battery, time, and task priorities
- **Decision Making**: Choose which actions to take (some save battery, others drain it)
- **Risk vs Reward**: Browsing Reddit costs 8% battery but only takes time

### 🎯 Real-World Learning
After each level, the game reveals **actual battery-saving tips** you can use in real life:

| Action | Game Effect | Real-World Impact |
|--------|-------------|-------------------|
| 关闭Chrome标签 | +3分钟电池寿命 | 每个标签消耗5-15%电量 |
| 降低屏幕亮度 | +5分钟电池寿命 | 屏幕占耗电的20-40% |
| 关闭后台应用 | +4分钟电池寿命 | 后台每小时耗10-15% |
| 开启省电模式 | +6分钟电池寿命 | 可延长2-3小时使用 |
| 拔掉USB设备 | +2分钟电池寿命 | USB设备持续取电 |

## 📝 Battery Tips Database

### 💻 Windows Tips
1. **电源模式**: 设置 → 系统 → 电源和电池 → 最佳电源效率
2. **后台应用**: 设置 → 应用 → 启动，关闭不必要的
3. **睡眠设置**: 缩短屏幕关闭和睡眠时间
4. **Edge效率模式**: 浏览器内置省电功能

### 🍎 Mac Tips
1. **低电量模式**: 系统设置 → 电池
2. **优化电池充电**: 延长电池寿命
3. **关闭动态桌面**: 静态壁纸更省电
4. **减少透明度**: 辅助功能 → 显示

### 🔧 Universal Tips
- **降低刷新率**: 60Hz比144Hz省电30%
- **关闭键盘背光**: 尤其是长时间打字
- **使用有线网络**: WiFi搜索网络会耗电
- **清理风扇**: 过热导致性能降低、耗电增加
