## 三、编程题（每题 25 分，共 50 分）

### 第 1 题：矩阵中的最长递增路径（25分）

**【问题描述】**

给定一个 n×m 的整数矩阵，找出其中最长的递增路径的长度。

递增路径是指：从某个单元格出发，每次可以移动到上下左右相邻的单元格，且下一个单元格的值必须大于当前单元格的值。路径的长度是路径上经过的单元格数量。

**【输入格式】**

第一行两个整数 n 和 m（1 ≤ n, m ≤ 100），表示矩阵的行数和列数。  
接下来 n 行，每行 m 个整数，表示矩阵中的元素。矩阵中元素的绝对值不超过 10⁴。

**【输出格式】**

一个整数，表示最长递增路径的长度。

**【样例输入】**

```
3 3
9 9 4
6 6 8
2 1 1
```

**【样例输出】**

```
4
```

**【解释】**

最长递增路径为：1 → 2 → 6 → 9，长度为 4。

### 第 2 题：最小花费爬楼梯（25分）

**【问题描述】**

某楼梯共有 n 级台阶，从第 0 级开始。每次可以爬 1 级或 2 级台阶。每级台阶都有一个花费值，表示到达该台阶所需的花费（注意：第 0 级和第 1 级的花费可以认为都是 0）。

你可以从第 0 级或第 1 级开始爬楼梯。当你在第 i 级时，你可以花费 `cost[i]` 向上爬 1 级到第 i+1 级，或者向上爬 2 级到第 i+2 级。

求到达楼梯顶部的最小花费。（楼梯顶部是指第 n 级，在数组之外）

**【输入格式】**

第一行一个整数 n（2 ≤ n ≤ 1000），表示楼梯的台阶数。  
第二行 n 个整数，表示每级台阶的花费值 `cost[i]`（0 ≤ cost[i] ≤ 999）。

**【输出格式】**

一个整数，表示到达楼梯顶部的最小花费。

**【样例输入 1】**

```
3
10 15 20
```

**【样例输出 1】**

```
15
```

**【解释 1】**

最小花费方案：从第 1 级开始（花费 0），直接爬 2 级到达顶部。总花费 = 0。

**【样例输入 2】**

```
10
1 100 1 1 1 100 1 1 100 1
```

**【样例输出 2】**

```
6
```

**【解释 2】**

最优路径：0 → 2 → 3 → 4 → 5 → 7 → 8 → 10，经过的花费为 1+1+1+1+1+1 = 6。
