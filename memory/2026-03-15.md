### Key Technical Decision: Draw Mode Logic
Final implementation:
```javascript
// On touch start, determine mode based on initial cell state
drawMode = isCellAlive(startX, startY) ? 'erase' : 'create'

// During drag, apply consistent mode
drawLine(fromX, fromY, toX, toY) {
  if (drawMode === 'create') setCell(x, y, true)
  else setCell(x, y, false)
}
```

This prevents mixed create/erase behavior during a single drag gesture.
