## WinControl Prototype Built

Built browser-based remote desktop control system for Windows:

**Features:**
- ✅ Full desktop capture at ~7 FPS via WebSocket
- ✅ Click anywhere to control that spot on PC
- ✅ No cursor jumping (uses absolute coordinates)
- ✅ Minimal UI — just the desktop, full screen

**Technical:**
- Python server: websockets, Pillow, mss, pywin32
- HTML/JS client: pure desktop view, no chrome
- Ratio-based coordinate mapping (click position → screen position)
- Desktop capture (no window selection needed)

**Usage:**
```bash
cd projects/win-control
pip install -r requirements.txt
python server.py
# Open http://localhost:8766
```

**Location:** `projects/win-control/`
