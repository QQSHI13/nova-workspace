## 算法复习笔记

### 一、状压 DP (State Compression DP)

**适用场景**: 状态可以用二进制表示的问题，通常 n ≤ 20

**核心思想**: 用二进制数的每一位表示一个元素的状态（选/不选、访问/未访问等）

#### 模板代码
```cpp
// dp[mask] 表示状态为 mask 时的最优值
int dp[1 << n];  // 2^n 种状态

// 初始化
dp[0] = 0;
for (int mask = 1; mask < (1 << n); mask++) dp[mask] = INF;

// 状态转移
for (int mask = 0; mask < (1 << n); mask++) {
    for (int i = 0; i < n; i++) {
        if (!(mask & (1 << i))) {  // 第 i 个元素未选
            int new_mask = mask | (1 << i);
            dp[new_mask] = min/max(dp[new_mask], dp[mask] + cost);
        }
    }
}
```

#### 常用位运算技巧
| 操作 | 代码 | 含义 |
|-----|------|------|
| 判断第 i 位是否为 1 | `mask & (1 << i)` | 检查状态 |
| 将第 i 位置 1 | `mask | (1 << i)` | 添加元素 |
| 将第 i 位置 0 | `mask & ~(1 << i)` | 移除元素 |
| 切换第 i 位 | `mask ^ (1 << i)` | 翻转状态 |
| 枚举所有子集 | `for (int s = mask; s; s = (s-1) & mask)` | 子集枚举 |

#### 经典例题
1. **旅行商问题 (TSP)** - 最短 Hamilton 路径
2. **集合覆盖问题** - 最少集合覆盖所有元素
3. **铺砖问题** - 用砖块铺满网格的方案数

### 二、区间 DP (Interval DP)

**适用场景**: 解决区间上的最优化问题，通常通过合并小区间解决大区间

**核心思想**: 
```
dp[i][j] = 区间 [i,j] 上的最优解
        = 对所有分割点 k: min/max(dp[i][k] + dp[k+1][j] + cost)
```

#### 模板代码
```cpp
// 初始化：长度为1的区间
for (int i = 1; i <= n; i++) dp[i][i] = 0;

// 按区间长度从小到大枚举
for (int len = 2; len <= n; len++) {          // 区间长度
    for (int i = 1; i + len - 1 <= n; i++) {  // 起点
        int j = i + len - 1;                   // 终点
        dp[i][j] = INF;  // 初始化为极大值
        
        // 枚举分割点
        for (int k = i; k < j; k++) {
            dp[i][j] = min(dp[i][j], dp[i][k] + dp[k+1][j] + cost);
        }
    }
}
// 答案：dp[1][n]
```

#### 时间复杂度
- **三层循环**: O(n³) — 长度 × 起点 × 分割点
- **优化**: 四边形不等式优化可降至 O(n²)

#### 经典例题
1. **石子合并** - 相邻合并最小代价
2. **矩阵链乘** - 最少乘法次数
3. **括号匹配** - 最少添加括号数
4. **回文串** - 最长回文子序列

#### 记忆口诀
```
区间DP三要素：
1. 状态 dp[i][j] —— 表示区间[i,j]
2. 枚举 len —— 从小到大枚举长度
3. 枚举 k —— 在[i,j-1]中找分割点
```

## 关联题目总结

### 前缀异或 + 状态压缩 (新题型)
这两道题都是利用**前缀状态相同**的性质：

| 题目 | 状态表示 | 判断条件 |
|-----|---------|---------|
| P10724 区间乘积 | 质因数指数奇偶性 | `bin[r] == bin[l-1]` |
| P11965 等价消除 | 字符次数奇偶性 | `mask[r] == mask[l-1]` |

**通用套路**:
```
1. 找性质：区间[l,r]满足某条件
2. 想前缀：条件转化为 prefix[r] == prefix[l-1]
3. 状压/哈希：快速比较前缀状态
4. 统计对数：用哈希表统计相同状态出现次数
```

**Floyd**
```cpp
for (int k = 1; k <= n; k++)
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= n; j++)
            dp[i][j] = min(dp[i][j], dp[i][k] + dp[k][j]);
```

**BFS (图论)**
```cpp
vector<int> adj[N];
int dist[N];
void bfs(int s) {
    queue<int> q;
    memset(dist, -1, sizeof(dist));
    dist[s] = 0; q.push(s);
    while (!q.empty()) {
        int u = q.front(); q.pop();
        for (int v : adj[u]) if (dist[v] == -1) {
            dist[v] = dist[u] + 1;
            q.push(v);
        }
    }
}
```
