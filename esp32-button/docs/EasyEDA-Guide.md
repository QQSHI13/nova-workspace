```
┌─────────────────────────────────────┐
│  USB-C 母座 (J1)                    │
│  VBUS → TP4056 VIN                  │
│  GND  → TP4056 GND                  │
└─────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────┐
│  TP4056 (U1) - SOP-8                │
│  • VIN: USB VBUS                    │
│  • GND: GND                         │
│  • BAT: 电池正极 (经过保护电路)     │
│  • PROG: 1.2kΩ 电阻到 GND          │
│  • CHRG: 红色 LED (1kΩ 限流)       │
│  • STDBY: 绿色 LED (1kΩ 限流)      │

```

### 步骤 2: 电池保护电路 (可选但推荐)

```
┌─────────────────────────────────────┐
│  DW01A (U2) + FS8205A (U3)          │
│  保护功能:                          │
│  • 过充保护 (>4.2V)                │
│  • 过放保护 (<2.4V)                │
│  • 过流保护                        │
│  • 短路保护                        │
└─────────────────────────────────────┘
```

### 步骤 3: LDO 稳压

```
┌─────────────────────────────────────┐
│  XC6206P332MR (U4) - SOT-23         │
│  • VIN: 电池输出 (3.7V)            │
│  • VOUT: 3.3V (连接到 GROVE 5V 脚) │
│  • GND: GND                         │
│  • 旁路电容: 10µF (输入/输出各一个)│
└─────────────────────────────────────┘
```

### 步骤 4: GROVE 连接器

```
                    AtomS3 Lite
                       │
    ┌──────────────────┼──────────────────┐
    │                  │                  │
    ▼                  ▼                  ▼
  GND(黑)           5V(红)             G1(黄)
    │                  │                  │
    └──────────────────┴──────────────────┘
              HY2.0-4P 连接器
```

**引脚分配:**
- Pin 1 (黑): GND
- Pin 2 (红): 5V (实际 3.3V 从 LDO 输出)
- Pin 3 (黄): G1 → 蜂鸣器
- Pin 4 (白): G32 (备用，可不连接)

## PCB 布局要点

### 板尺寸与安装方式
- **大小**: 24.0 x 24.0 mm (与 AtomS3 Lite 相同)
- **安装**: **背包式** (Backpack) - 扩展板位于 AtomS3 Lite 下方
- **连接**: GROVE 4P 排线 + 中心 M2 螺丝固定
- **层数**: 2 层
- **板厚**: 1.6mm
- **铜厚**: 1oz

### 背包式设计说明
```
AtomS3 Lite (顶层，按钮朝上)
    │
    │ GROVE 4P 排线
    ▼
扩展板 (底层，元件朝下)
    │
    │ M2 螺丝 (中心)
    ▼
电池 (最底层)
```

### 关键规则

#### 1. M2 安装孔 (背包式安装)
```
位置: 中心 (12mm, 12mm)
孔径: 2.2mm (M2 螺丝)
禁止布线区: 6mm 直径
说明: 扩展板位于 AtomS3 Lite 下方，通过中心螺丝固定
```

#### 2. 元件布局 (从上到下)

```
┌──────────────────────────────┐
│     [USB-C 母座]             │ ← 顶部边缘
│                              │
│  [TP4056] [DW01A] [FS8205A]  │ ← 充电管理
│                              │
│  [红LED] [绿LED]             │ ← 状态指示
│                              │
│        [M2 孔]               │ ← 中心安装孔
│                              │
│  [XC6206] [蜂鸣器]           │ ← 稳压 + 反馈
│                              │
│  [JST-PH] [滑动开关]         │ ← 电池 + 开关
│                              │
│     [HY2.0-4P]               │ ← 底部边缘 (连接 Atom)
└──────────────────────────────┘
```

#### 3. 走线规则

**电源线 (粗):**
- USB 5V → TP4056: 0.6mm
- 电池 → 保护电路: 0.6mm
- LDO 输出 → GROVE: 0.5mm

**信号线 (细):**
- LED 控制: 0.25mm
- 蜂鸣器: 0.3mm

**GND 处理:**
- 底层全部铺铜作为 GND
- 顶层局部铺铜
- 多处过孔连接上下层 GND

#### 4. 关键距离

```
USB-C 母座: 贴近板边缘 (方便插线)
JST-PH: 距离边缘 2mm 以上
HY2.0-4P: 贴近底部边缘 (对接 AtomS3 Lite)
```

## 布线步骤

### 第 1 步: 放置元件
1. 先放连接器 (USB-C, HY2.0-4P, JST-PH) 在边缘
2. 放置大元件 (蜂鸣器, 开关)
3. 放置 IC (TP4056, XC6206 等)
4. 放置小元件 (电阻电容 LED)

### 第 2 步: 电源布线
1. USB VBUS → TP4056 VIN
2. TP4056 BAT → 保护电路 → 电池
3. 电池 → XC6206 VIN
4. XC6206 VOUT → HY2.0-4P Pin 2 (5V 脚)

### 第 3 步: 信号布线
1. TP4056 CHRG/STDBY → LED (经过限流电阻)
2. HY2.0-4P Pin 3 (G1) → 蜂鸣器
3. 所有 GND 连接到铺铜

### 第 4 步: 铺铜
1. 底层: 完整 GND 铺铜
2. 顶层: 局部 GND 铺铜 (避开信号线)

## DRC 检查清单

提交 JLCPCB 前检查:

- [ ] 最小线宽 ≥ 0.15mm
- [ ] 最小间距 ≥ 0.15mm
- [ ] 过孔尺寸: 0.6mm 外径 / 0.3mm 内径
- [ ] 丝印不重叠
- [ ] 安装孔位置正确
- [ ] 连接器方向正确

## 导出 Gerber

1. 点击 **文件 → 导出 → Gerber**
2. 设置:
   - 层: 全部
   - 格式: RS-274X
   - 包含钻孔文件: 是
3. 压缩为 ZIP 文件

## JLCPCB 下单

1. 访问 https://jlcpcb.com/
2. 上传 Gerber ZIP
3. 设置:
   - 板材: FR-4
   - 层数: 2
   - 尺寸: 24 x 24 mm
   - 数量: 5
   - 颜色: 黑色 (推荐)
   - 表面处理: 沉金 (ENIG) - 可选
   - 铜厚: 1oz
4. 保存到购物车

## 常见问题

**Q: 为什么 LDO 输出接 GROVE 的 5V 脚？**
A: AtomS3 Lite 的 5V 脚可以接受 3.3V-5V 输入，我们提供稳压后的 3.3V 是为了延长电池寿命。

**Q: 可以不用电池保护芯片吗？**
A: 可以，但强烈推荐使用。DW01A+FS8205A 可以防止过充/过放导致电池损坏甚至起火。

**Q: 蜂鸣器可以直接接 GPIO 吗？**
A: 可以，GPIO26 可以直接驱动小型蜂鸣器。如需更响的声音，可以加一个三极管驱动。
